# How to Implement Laya Batterymonitor into my vendor?
1. Place vendor.azenith-service binary to /vendor/bin/hw/
2. Place init.azenith.rc to /vendor/etc/init/
3. Patch vendor sepolicy cil
4. Repack Vendor
